# P2 Roadmap

## P2.0: 底座骨架

- 会话消息模型
- AI 建议模型
- OCR 结果模型
- 同步记录模型
- Repository / Service 合同
- 本地 Stub 实现
- SQLite 表与 QA 扩展

说明：
- 本轮不是 AI / OCR / 自动回复 / 云同步的真实实现。

## P2.1: 聊天消息手工录入 / 导入

- 手工录入消息
- 按客户 / 订单查看消息
- 为后续 AI 建议提供上下文

## P2.2: AI 回复建议真实接入

- 接入真实 AI Provider
- 提示词和上下文裁剪
- 建议生成、接受、拒绝、发送状态闭环

## P2.3: OCR 截图识别

- 接入真实 OCR
- 支持截图 / 文件导入
- 回填识别文本与失败原因

## P2.4: 自动回复半自动化

- 人工确认后发送
- 发送记录留痕
- 平台边界与权限控制

## P2.5: 云同步 / 备份

- 本地到云端备份
- 最小冲突处理
- 恢复与状态可见性

## 边界声明

- AI 不是 P2.0 真实实现。
- OCR 不是 P2.0 真实实现。
- 自动回复不是 P2.0 真实实现。
- 云同步不是 P2.0 真实实现。
