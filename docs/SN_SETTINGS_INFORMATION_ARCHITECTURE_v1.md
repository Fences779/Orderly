# SN 设置页信息架构树 v1

> 定位：设置页只服务「应用本身」与「SN 接入能力」，不做店铺装修、不做普通后台资料页。

```text
设置
├─ 1. 应用设置
│  ├─ 启动行为
│  │  ├─ 启动时默认打开页面：工作台 / 订单履约 / 异常处理 / 客户档案
│  │  ├─ 是否记住上次打开页面
│  │  └─ 是否开机自启动
│  ├─ 窗口与显示
│  │  ├─ 记住窗口大小与位置
│  │  ├─ 默认窗口模式：普通 / 最大化
│  │  ├─ 侧边栏默认展开 / 收起
│  │  ├─ 字体大小：小 / 标准 / 大
│  │  └─ 界面缩放提示：跟随 Windows 缩放
│  └─ 主题与外观
│     ├─ 主题模式：浅色 / 深色 / 跟随系统
│     ├─ 强调色：默认绿 / 茶金 / 雾蓝
│     └─ 是否开启轻动画
│
├─ 2. 数据与备份
│  ├─ 本地数据
│  │  ├─ 当前数据库路径
│  │  ├─ 打开数据目录
│  │  ├─ 数据库大小
│  │  └─ 数据健康状态
│  ├─ 备份设置
│  │  ├─ 备份目录
│  │  ├─ 自动备份开关
│  │  ├─ 自动备份频率：每日 / 每周 / 手动
│  │  ├─ 保留最近 N 份备份
│  │  └─ 立即备份
│  ├─ 导入导出
│  │  ├─ 导出订单数据
│  │  ├─ 导出客户档案
│  │  ├─ 导出操作日志
│  │  └─ 导入历史数据
│  └─ 清理维护
│     ├─ 清理缓存
│     ├─ 清理过期日志
│     └─ 数据完整性检查
│
├─ 3. SN 接入设置
│  ├─ 云开发连接
│  │  ├─ 云开发环境 ID
│  │  ├─ 当前连接状态
│  │  ├─ 最近连接时间
│  │  └─ 测试连接
│  ├─ 订单同步
│  │  ├─ 订单同步开关
│  │  ├─ 同步模式：手动 / 定时
│  │  ├─ 同步频率
│  │  ├─ 最近同步时间
│  │  ├─ 最近同步结果
│  │  └─ 手动同步订单
│  ├─ 同步日志
│  │  ├─ 查看最近同步日志
│  │  ├─ 查看失败记录
│  │  └─ 导出同步日志
│  └─ 只读保护提示
│     ├─ 支付回调链路：只读展示，不可改动
│     ├─ paid 状态写入链路：只读展示，不可改动
│     ├─ 微信发货同步链路：只读展示，不可改动
│     └─ 履约回传链路：只读展示，不可改动
│
├─ 4. 安全与隐私
│  ├─ 本地安全
│  │  ├─ 数据库加密状态
│  │  ├─ 备份文件加密状态
│  │  └─ 本机访问保护
│  ├─ 敏感信息
│  │  ├─ 手机号默认脱敏
│  │  ├─ 地址默认脱敏
│  │  ├─ 导出时是否包含敏感字段
│  │  └─ 复制订单摘要时是否脱敏
│  ├─ 操作日志
│  │  ├─ 操作日志开关
│  │  ├─ 日志保留周期
│  │  ├─ 查看操作日志
│  │  └─ 清理操作日志
│  └─ 风险提示
│     ├─ 明文数据库风险提示
│     ├─ 明文备份风险提示
│     └─ AI 外发上下文风险提示
│
├─ 5. AI 设置
│  ├─ AI 功能开关
│  │  ├─ 是否启用 AI 辅助
│  │  ├─ 是否允许读取订单上下文
│  │  └─ 是否允许读取客户档案上下文
│  ├─ 模型配置
│  │  ├─ 默认模型
│  │  ├─ API Key 状态
│  │  ├─ 测试 AI 连接
│  │  └─ 模型调用超时时间
│  ├─ 内容安全
│  │  ├─ 发送前自动脱敏
│  │  ├─ 禁止发送手机号
│  │  ├─ 禁止发送完整地址
│  │  └─ 禁止发送支付流水号
│  └─ 输出偏好
│     ├─ 回复语气：简洁 / 温和 / 专业
│     ├─ 回复长度：短 / 标准 / 详细
│     └─ 是否自动生成订单摘要
│
├─ 6. 快捷键
│  ├─ 全局快捷键
│  │  ├─ 打开 / 隐藏主窗口
│  │  ├─ 快速搜索订单
│  │  └─ 打开今日工作台
│  ├─ 订单快捷键
│  │  ├─ 复制当前订单摘要
│  │  ├─ 打开制作单
│  │  ├─ 标记异常
│  │  └─ 推进履约状态
│  └─ 客户快捷键
│     ├─ 打开客户档案
│     ├─ 新建客户备注
│     └─ 复制客户偏好摘要
│
├─ 7. 通知与提醒
│  ├─ 桌面通知
│  │  ├─ 新订单提醒
│  │  ├─ 异常订单提醒
│  │  ├─ 超时未处理提醒
│  │  └─ 同步失败提醒
│  ├─ 提醒规则
│  │  ├─ 已支付 N 小时未确认
│  │  ├─ 待制作 N 小时未推进
│  │  ├─ 待发货 N 小时未发货
│  │  └─ 地址缺失立即提醒
│  └─ 免打扰
│     ├─ 免打扰时间段
│     └─ 是否仅显示高优先级提醒
│
└─ 8. 关于与诊断
   ├─ 应用信息
   │  ├─ 当前版本
   │  ├─ 构建时间
   │  ├─ 本机环境
   │  └─ 更新检查
   ├─ 诊断工具
   │  ├─ 复制诊断信息
   │  ├─ 导出错误日志
   │  ├─ 打开日志目录
   │  └─ 重启应用
   └─ 开发者/调试
      ├─ Debug 模式开关
      ├─ QA 数据入口
      └─ 重置本地测试状态
```

## 第一版落地优先级

```text
P0 必做
├─ 应用设置
├─ 数据与备份
├─ SN 接入设置
├─ 安全与隐私
└─ 关于与诊断

P1 再做
├─ AI 设置
├─ 快捷键
└─ 通知与提醒

暂不做
└─ 店铺资料 / 店铺装修 / 营销配置
```
